
# :rocket: WebCodeTests

Bem-vindo ao meu repositório de prática de desenvolvimento Frontend. Este espaço é dedicado ao aprimoramento de habilidades em HTML, CSS e JavaScript (ou frameworks), através da construção de interfaces modernas e responsivas.

## :bulb: Sobre o Projeto

Diferente de resoluções literais, os projetos contidos aqui são autorais e inspirados em propostas de plataformas renomadas como:

* [Frontend Mentor](https://www.frontendmentor.io/)
* [Frontend Practice](https://www.frontendpractice.com/)
* [DevChallenges](https://devchallenges.io/)

O objetivo é capturar a essência visual e funcional desses desafios, mas aplicando soluções próprias, toques de design pessoal e explorando diferentes tecnologias de mercado.

## :hammer_and_wrench: Tecnologias Utilizadas

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Estilização:** Tailwind CSS / Sass / Styled 
- **ComponentsDeploy:** Vercel / GitHub Pages

## :chart_with_upwards_trend: Evolução Técnica

Ao longo desses desafios, venho consolidando conceitos fundamentais para o mercado:
1. **Metodologia Mobile First:** Garantindo que a aplicação seja leve e funcional em telas menores antes de escalar para desktop.
2. **Acessibilidade (a11y):** Uso de HTML semântico e atributos ARIA para uma web mais inclusiva.
3. **Clean Code:** Estruturação de pastas lógica e código de fácil manutenção.
4. **Performance:** Otimização de imagens e carregamento de assets.
